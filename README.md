# Blog

Built using [Next.js](https://nextjs.org) and deployed on [Vercel](https://vercel.com).

## Run locally

To run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
